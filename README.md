## Sobre el proyecto



Este proyecto se basa en un modelo cliente-servidor en el que el cliente (escrito en Java) genera una GUI para el usuario introducir datos referentes a proyectos en una empresa audiovisual.
El servidor escrito en Node.js recibe las peticiones para resolverlas y enviarlas al cliente.

## Introducción

### Prerequisitos

Instalar las librerias de node usadas con
  ```sh
  npm install
  ```
Iniciar el servidor en /backend
  ```sh
  npm run start
  ```
